# Citistek

A modern React + TypeScript application cloning the Anduril.com homepage design.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- CSS3
